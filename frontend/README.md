# SocialMap Frontend

Basit ve sade bir React front-end uygulaması.

## Kurulum

```bash
npm install
```

## Çalıştırma

```bash
npm run dev
```

Uygulama `http://localhost:5173` adresinde çalışacaktır.

## Özellikler

- **Gönderiler**: Gönderileri görüntüleme, oluşturma, beğenme ve yorum yapma
- **Kullanıcılar**: Kullanıcıları görüntüleme ve yeni kullanıcı oluşturma
- **Yerler**: Yerleri görüntüleme, arama ve yeni yer ekleme

## Backend API

Backend API'nin `http://localhost:5280` adresinde çalıştığından emin olun.
